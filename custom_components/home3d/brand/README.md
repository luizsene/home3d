Place your Home Assistant brand images for the home3d integration in this folder.

Required minimum files:
- icon.png (square, 256x256)
- logo.png (recommended)

Optional files:
- dark_icon.png
- dark_logo.png
- icon@2x.png
- logo@2x.png
- dark_icon@2x.png
- dark_logo@2x.png

Notes:
- PNG format only.
- If logo.png is missing, Home Assistant falls back to icon.png.
- Restart Home Assistant after adding/replacing images.
